# minion-speak-app
A translation web app that will translate your sentences into banana language of the minions.

## Technologies used:
1. HTML
1. CSS
1. VanillaJS
1. Fetch API
1. Minion Speak Translator from Fun Translations API - https://funtranslations.com/api/minion
